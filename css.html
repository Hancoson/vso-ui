栅格 grid

TreeBrer UI 提供了一套响应式、PC设备优先的流式栅格系统，随着屏幕或视口（viewport）尺寸的减小，系统会自动分为最多12列。栅格系统用于通过一系列的行（row）与列（column）的组合来创建页面布局，你的内容就可以放入这些创建好的布局中。

1.媒体查询

在栅格系统中，我们在 Less 文件中使用以下媒体查询（media query）来创建关键的分界点阈值。

/* 超大屏幕（大于 1200px） */
/* 没有任何媒体查询相关的代码，因为这在 TreeBrer UI 中是默认的（还记得 TreeBrer UI 是PC备优先的吗？） */

/* 大屏幕（大桌面显示器，小于等于 1200px） */
@media (max-width: @screen-lg-max) { ... }

/* 中等屏幕（桌面显示器，小于等于 960px） */
@media (max-width: @screen-md-max) { ... }

/* 小屏幕（平板，小于等于 768px） */
@media (max-width: @screen-sm-max) { ... }

/* 手机屏幕（手机，小于等于 640px） */
@media (max-width: @screen-mm-max) { ... }

按钮 button

1.基本使用

1.1 默认样式

在要应用按钮样式的元素上添加 .tbui-btn，再设置相应的颜色。

  • .tbui-btn-default - 默认，灰色按钮
  • .tbui-btn-primary - 蓝色按钮
  • .tbui-btn-secondary - 浅蓝色按钮
  • .tbui-btn-success - 绿色按钮
  • .tbui-btn-warning - 橙色按钮
  • .tbui-btn-danger - 红色按钮
  • .tbui-btn-link- 链接

表单 forms

1.基本实例

单独的表单控件会被自动赋予一些全局样式。所有设置了 .form-control 类的 <input>、<textarea> 和 <select> 元素都将被默认设置宽度属性为 width: 100%;。 将 label 元素和前面提到的控件包裹在 .form-group 中可以获得最好的排列。

2.表单排列

2.1 水平排列

在 <form> 添加 .tbui-form-horizontal class 并结合网格系统使用。
